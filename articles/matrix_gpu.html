Matrix Addition Performance

We compare how fast a CPU and a GPU can perform a simple matrix addition for a 512x512 matrix.

CPU Code

#define MATRIX_SIZE 512
for (int i = 0; i < MATRIX_SIZE; ++i) {
    for (int j = 0; j < MATRIX_SIZE; ++j) {
        A[i][j] = static_cast<float>(i + j);
        B[i][j] = static_cast<float>(i - j);
    }
}
CPU Time: 0.000448083 seconds

GPU Code (Metal)

#define MATRIX_SIZE 512
kernel void matrix_add(device const float* A[[buffer(0)]],
                       device const float* B[[buffer(1)]],
                       device float* C[[buffer(2)]],
                       uint id [[thread_position_in_grid]]) {
    uint row = id / MATRIX_SIZE;
    uint col = id % MATRIX_SIZE;
    C[id] = A[id] + B[id];
}
GPU Time: 0.000044 seconds

Conclusion

The GPU outperformed the CPU significantly in this task, demonstrating its advantage in parallel computations.
